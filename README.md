# 🌍 Tourism Explorer India

A full-stack web application that allows users to explore popular tourist destinations across India with nearby attractions.

## 🚀 Features
- 🔐 User Authentication (Login & Register)
- 🗺️ Explore Top Tourist Destinations
- 📍 View Nearby Places for Each Destination
- 📖 "More Info" feature with descriptions
- 👥 Tracks number of users (SQLite Database)
- 🎨 Modern UI with animations & slideshow background

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite
- Version Control: Git & GitHub
